# Scripts d'Intelligence Artificielle

Scripts pour l'entraînement et la gestion des modèles IA de traduction.

## Scripts disponibles

### `train-translation-model.sh`
Entraîne le modèle de traduction à partir des résultats de test.

**Utilisation :**
```bash
make train-translation
```

### `ai-training-menu.sh`
Menu interactif complet pour le système d'entraînement IA.

**Utilisation :**
```bash
make train-ai
```

## Commandes Make associées

- `make retrain-ml` - Réentraîne le modèle ML avec tous les feedbacks approuvés
- `make translation-stats` - Affiche les statistiques des feedbacks
- `make update-translation-dict` - Met à jour les dictionnaires JSON
