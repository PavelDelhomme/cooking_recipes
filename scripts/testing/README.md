# Scripts de Test

Scripts pour tester l'application et l'API.

## Scripts disponibles

### `test_api.sh`
Teste l'API backend et la récupération de recettes.

**Utilisation :**
```bash
make test-api
```

### `test-recipes.sh`
Test interactif des recettes pour entraîner le modèle de traduction.

**Utilisation :**
```bash
make test-recipes [NUM_RECIPES]
```
